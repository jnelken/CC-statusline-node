## What does *your* terminal use?

Most people never change their terminal font, so this is probably what you're already seeing:

| OS / Terminal | Default monospace font |
|---------------|------------------------|
| **macOS** — Terminal.app | **SF Mono** (Big Sur and later) |
| **macOS** — VS Code terminal | **Menlo** |
| macOS — classic | Monaco |
| **Windows** — Windows Terminal (Win 11) | **Cascadia Mono** |
| **Windows** — PowerShell / cmd.exe | **Consolas** |
| Linux — GNOME Terminal | DejaVu Sans Mono |
| Powerlevel10k / oh-my-zsh | MesloLGS Nerd Font |

> The statusline looks great in all of them — pick whatever you already use.
> *(Consolas, Cascadia Mono, and DejaVu Sans Mono are OS-bundled fonts not installed on the capture machine, so the image substitutes the closest match — e.g. **Cascadia Code** for Windows Terminal.)*
